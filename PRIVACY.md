# Privacy

AdPages Tools for PHP is a local utility library.

## Data Handling

- The package does not make network calls.
- The package does not collect, store, transmit, or sell user data.
- The package does not use cookies, browser storage, telemetry, analytics, or a hosted backend.
- Inputs are processed in memory by the PHP process that imports the library.

## External Services

No external services are contacted by the library, examples, or local validation scripts.

## User Responsibility

Applications that import this library are responsible for their own privacy notices, consent flows, analytics tools, logging behavior, and data retention practices.
